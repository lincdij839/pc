# PC 语言开发计划

## 阶段 1：完善基础功能（当前阶段）

### 1.1 增强字符串操作
- [ ] 字符串切片 `s[1:5]`
- [ ] 字符串格式化 `f"Hello {name}"`
- [ ] 字符串方法 `upper()`, `lower()`, `split()`, `join()`

### 1.2 增强列表操作
- [ ] 列表切片 `list[1:5]`
- [ ] 列表方法 `append()`, `pop()`, `insert()`, `remove()`
- [ ] 列表推导式 `[x*2 for x in range(10)]`

### 1.3 增强字典操作
- [ ] 字典方法 `keys()`, `values()`, `items()`, `get()`
- [ ] 字典推导式 `{k: v*2 for k, v in dict.items()}`

### 1.4 文件操作
- [ ] 文件读写 `open()`, `read()`, `write()`, `close()`
- [ ] with 语句支持
- [ ] 文件迭代

### 1.5 错误处理
- [ ] try/except/finally 语法
- [ ] raise 语句
- [ ] 自定义异常类型

## 阶段 2：类型系统（下一阶段）

### 2.1 类型注解
- [ ] 变量类型注解 `x: int = 10`

- [ ] 类型检查器

### 2.2 类型推导
- [ ] 自动类型推导
- [ ] 类型兼容性检查

## 阶段 3：面向对象（未来）

### 3.1 类实现
- [ ] 类定义和实例化
- [ ] 构造函数 `__init__`
- [ ] 实例方法和属性
- [ ] 类方法和静态方法

### 3.2 继承
- [ ] 单继承
- [ ] 方法重写

## 当前优先级

1. **字符串操作增强** - 最常用
2. **文件操作** - 系统管理必需
3. **错误处理** - 提高健壮性
4. **列表/字典方法** - 提高易用性
